# 本项目为网易云音乐平台爬虫。

## 实现功能

根据关键字搜索歌曲，返回歌曲名称、歌手、id。

根据歌曲id下载歌曲为mp3格式，保存在本地，并将歌曲名称、歌手、专辑名称、专辑作者、专辑封面、歌词（带翻译）嵌入到mp3文件的元数据中。

由于作者无网易云音乐VIP，所以无法确认VIP用户的Cookie能否下载VIP歌曲，下载的歌曲的品质也不可选择。但是非VIP歌曲是可以下载的。

## 使用方法

安装依赖后，在源文件里设置Cookie，然后直接运行：

```shell
python main.py
python ui.py
```

二者之一即可，前者为命令行界面，后者为图形界面。
